# Icon

Aristid Icon is a on/off state icon

## Usage

##### How to import
```jsx
import KitIcon from "@kit/General/";
import { UserOutlined } from "@ant-design/icons";

<KitIcon icon={<UserOutlined />}/>
```

## Exemples

##### Basic

A basic Icon with on without `on` property.

<Canvas withSource="none">
    <KitSpace>
        <KitIcon icon={<DownloadOutlined/>} />
        <KitIcon icon={<DownloadOutlined/>} on />
    </KitSpace>
</Canvas>

##### Hoverable

Show Icon with `on` property on hover state.

<Canvas withSource="none">
    <KitSpace>
        <KitIcon icon={<DownloadOutlined/>} hoverable/>
        <KitIcon icon={<DownloadOutlined/>} hoverable on/>
    </KitSpace>
</Canvas>

##### Clickable

Icon can be clickable using `onClick` property.

<Canvas withSource="none">
    <KitSpace>
        <KitIcon icon={<DownloadOutlined/>} onClick={() => console.log("click")}/>
        <KitIcon icon={<DownloadOutlined/>} hoverable onClick={() => console.log("click")}/>
    </KitSpace>
</Canvas>
