# Divider

A divider line separates different content.

## Usage
-   Divide sections of article.
-   Divide inline text and links such as the operation column of table.

## Examples

##### Horizontal

Divider is `horizontal` by default. You can add text within Divider.

<CustomCanvas content={Exemples.Horizontal} source={Sources.Horizontal} id="divider-horizontal" />

##### Vertical

Use `type="vertical"` make it vertical..


<CustomCanvas content={Exemples.Vertical} source={Sources.Vertical} id="divider-vertical" />

##### No margin

Use `noMargin` remove top and bottom margin.


<CustomCanvas content={Exemples.NoMargin} source={Sources.NoMargin} id="divider-nomargin" />

##### Divider with title

Divider with inner title, set `orientation="left/right"` to align it.


<CustomCanvas content={Exemples.WithTitle} source={Sources.WithTitle} id="divider-title" />

##### Color

Use `color` to change border color.


<CustomCanvas content={Exemples.Color} source={Sources.Color} id="divider-color" />
