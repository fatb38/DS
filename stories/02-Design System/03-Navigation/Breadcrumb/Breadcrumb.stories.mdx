# Breadcrumb
A breadcrumb displays the current location within a hierarchy. It allows going back to states higher up in the hierarchy.

## Usage

- When the system has more than two layers in a hierarchy.
- When you need to inform the user of where they are.
- When the user may need to navigate back to a higher level.

##### How to import
```jsx
import { KitBreadcrumb } from "@kit/Navigation/";

<KitBreadcrumb ... />
```

### ItemType

> type ItemType = `RouteItemType` | `SeparatorType`

### RouteItemType

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| className | The additional css class | string | - |
| dropdownProps | The dropdown props | `Dropdown` | - |
| href | Target of hyperlink | string | - |
| menu | The menu props | `MenuProps` | - |
| onClick | Set the handler to handle click event | (e:MouseEvent) => void | - |
| title | item name | ReactNode | - |

### SeparatorType

```ts
const item = {
  type: 'separator', // Must have
  separator: '/',
};
```

| Property  | Description       | Type        | Default |
| --------- | ----------------- | ----------- | ------- |
| type      | Mark as separator | `separator` |         |
| separator | Custom separator  | ReactNode   | `/`     |
