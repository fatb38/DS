# Header

Header is displayed in top of the app. It allows a menu and app logo on the left, a lang switch and a user profile on the right. The content in between is free (eg. title, search, breadcrumb, actions, etc.)

## Usage

-   When you need to have a global header, providing shortcuts, informations and actions throughout the app.

### Header.Profile

| Property             | Description                                                                   | Type              | Default |
| -------------------- | ----------------------------------------------------------------------------- | ----------------- | ------- |
| className            | The additional css class                                                      | string            | -       |
| logo                 | Url of the logo to display (optional)                                         | string            | -       |
| alt                  | This attribute defines the alternative text describing the image (optional)   | string            | -       |
| tagProps             | tag to display (optional)                                                     | `IKitTag`         | -       |
| profileMenuCardProps | `KitHeaderIdCard` to display inside menu (optional). See KitIdCard fo details | `KitHeaderIdCard` | -       |
| profileCardProps     | `KitIdCard` to display (optional). See KitIdCard fo details                   | `IKitIdCard`      | -       |
| complement           | Complementary element to display on the right of profile block (optional)     | `ReactNode`       | -       |
| menu                 | The menu props                                                                | `ReactNode`       | -       |

```typescript
type KitHeaderIdCard = IKitIdCard & {onClick?: () => void};
```
