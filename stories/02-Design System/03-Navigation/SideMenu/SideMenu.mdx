# SideMenu

A versatile side menu for navigation.

## Usage

Navigation is an important part of any website, as a good navigation setup allows users to move around the site quickly and efficiently.

## Examples

### SideMenu

##### Basic

A basic SideMenu is just a container.

### SideMeuItemProps
SideMenuItem also accepts `data_*` and `aria-*` attributes

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| key | The item key (used to identify active item) | string | - |
| type | The type of item (`default` \| `separator`) | string | `default` |
| title | Title of item | `ReactNode` | - |
| complement |Complement element to display after the titel when menu is open | `ReactNode` | - |
| icon | Icon to associate to item | `ReactNode` | - |
| disabled | whether the item is isabled | boolean | - |
| disabledSecureClick | whether to disable secure click on this particular item | boolean | - |
| onClick | Set the handler to handle click event (hthe `key`param is the key prop of the item|  (key: string) => void | - |
