# InputNumber

Enter a number within certain range with the mouse or keyboard.

## Usage

When a numeric value needs to be provided.

## Examples

##### Basic

Numeric-only input box.

<CustomCanvas content={Examples.Basic} source={Sources.Basic} id="inputnumber-basic" />

##### Disabled

Click the button to toggle between available and disabled states.

<CustomCanvas content={Examples.Disabled} source={Sources.Disabled} id="inputnumber-disabled" />

##### High precision decimals

Use `stringMode` to support high precision decimals support. `onChange` will return string value instead. You need polyfill of BigInt if browser not support.

<CustomCanvas content={Examples.Precision} source={Sources.Precision} id="inputnumber-precision" />

##### Keyboard

Control keyboard behavior by `keyboard`.

<CustomCanvas content={Examples.Keyboard} source={Sources.Keyboard} id="inputnumber-keyboard" />

##### Formatter

Display value within it's situation with `formatter`, and we usually use `parser` at the same time.

<CustomCanvas content={Examples.Formatter} source={Sources.Formatter} id="inputnumber-formatter" />

##### Prefix

Add an icon prefix inside input.

<CustomCanvas content={Examples.Prefix} source={Sources.Prefix} id="inputnumber-prefix" />

##### Out of range

Show warning style when `value` is out of range by control.

<CustomCanvas content={Examples.Outrange} source={Sources.Outrange} id="inputnumber-out-range" />

##### Status

Add status to InputNumber with `status`, which could be `error` or `warning`.

<CustomCanvas content={Examples.Status} source={Sources.Status} id="inputnumber-status" />

##### Label and Helper

Add `label` and/or `helper` properties to give field context.

<CustomCanvas content={Examples.Wrapper} source={Sources.Wrapper} id="inputnumber-wrapper" />
