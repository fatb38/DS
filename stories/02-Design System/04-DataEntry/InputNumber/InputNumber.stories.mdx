# InputNumber

Enter a number within certain range with the mouse or keyboard.

## Usage

When a numeric value needs to be provided.

## Examples

##### Basic

Numeric-only input box.

<CustomCanvas content={Exemples.Basic} source={Sources.Basic} />

##### Disabled

Click the button to toggle between available and disabled states.

<CustomCanvas content={Exemples.Disabled} source={Sources.Disabled} />

##### High precision decimals

Use `stringMode` to support high precision decimals support. `onChange` will return string value instead. You need polyfill of BigInt if browser not support.

<CustomCanvas content={Exemples.Precision} source={Sources.Precision} />

##### Keyboard

Control keyboard behavior by `keyboard`.

<CustomCanvas content={Exemples.Keyboard} source={Sources.Keyboard} />

##### Prefix

Add a prefix inside input.

<CustomCanvas content={Exemples.Prefix} source={Sources.Prefix} />

##### Out of range

Show warning style when `value` is out of range by control.

<CustomCanvas content={Exemples.Outrange} source={Sources.Outrange} />

##### Status

Add status to InputNumber with `status`, which could be `error` or `warning`.

<CustomCanvas content={Exemples.Status} source={Sources.Status} />

##### Label and Helper

Add `label` and/or `helper` properties to give field context.
