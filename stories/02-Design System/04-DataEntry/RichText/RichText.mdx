# RichText

This components is based on [Tiptap Editor](https://tiptap.dev/docs/editor/getting-started/overview).

## Usage

When user needs to format text, add links, and more.

## Examples

### Basic

Basic usage example. Set `defaultValue` property to provide the initial content for the editor.

<CustomCanvas content={Examples.Basic} source={Sources.Basic} id="richtext-basic" />

### Disabled

Set to disabled state.

<CustomCanvas content={Examples.Disabled} source={Sources.Disabled} id="richtext-disabled" />

### Set Value

Set `value` to provide the content for the editor (this value can be dynamic). You can also retrieve the value from the editor using `onChange` property.

<CustomCanvas content={Examples.SetValue} source={Sources.SetValue} id="richtext-set-value" />

### With character counting

Show character counting with `showCount` property. You can also set the maximum number of characters with `maxCount` property.

<CustomCanvas content={Examples.CharacterCount} source={Sources.CharacterCount} id="richtext-character-count" />

### Status

Add status to RichText with `status`, which could be `error` or `warning`.

<CustomCanvas content={Examples.Status} source={Sources.Status} id="richtext-status" />

### Label and Helper

Add `label` and/or `helper` properties to give context.

<CustomCanvas content={Examples.Wrapper} source={Sources.Wrapper} id="richtext-wrapper" />

### Read Only

Set to read only state.

<CustomCanvas content={Examples.ReadOnly} source={Sources.ReadOnly} id="richtext-read-only" />
