# Slider

Slider Component.

## Examples

##### Basic
Basic slider. When `range` is `true`, display as dual thumb mode. When `disable` is `true`, the slider will not be interactable.

<CustomCanvas content={Exemples.Basic} source={Sources.Basic} id="slider-basic" />

##### Slider with InputNumber
Synchronize with KitInputNumber component.

<CustomCanvas content={Exemples.WithInputNumber} source={Sources.WithInputNumber} id="slider-input" />

##### Slider with icon
You can add an icon beside the slider to make it meaningful.

<CustomCanvas content={Exemples.WithIcon} source={Sources.WithIcon} id="slider-icon" />

##### Customize tooltip
Use `tooltip.formatter` to format content of `Tooltip`. If `tooltip.formatter` is null, hide it.

<CustomCanvas content={Exemples.Formatter} source={Sources.Formatter} id="slider-formatter" />

##### Event
The `onChange` callback function will fire when the user changes the slider's value. The `onAfterChange` callback function will fire when `onmouseup` fired.

<CustomCanvas content={Exemples.Events} source={Sources.Events} id="slider-events" />

##### Reverse
Using `reverse` to render slider reversely.

<CustomCanvas content={Exemples.Reversed} source={Sources.Reversed} id="slider-reversed" />

##### Graduated slider
Using `marks` property to mark a graduated slider, use `value` or `defaultValue` to specify the position of thumb. When `included` is false, means that different thumbs are coordinative. when `step` is null, users can only slide the thumbs onto marks.

<CustomCanvas content={Exemples.Graduated} source={Sources.Graduated} id="slider-graduated" />

##### Control visible of ToolTip
When `tooltip.open` is `true`, ToolTip will always show, or ToolTip will not show anyway, even if dragging or hovering.

<CustomCanvas content={Exemples.VisibleTooltip} source={Sources.VisibleTooltip} id="slider-visible-tooltip" />

##### Draggable track
Make range track draggable when set `range.draggableTrack`.

<CustomCanvas content={Exemples.Draggable} source={Sources.Draggable} id="slider-draggable" />
