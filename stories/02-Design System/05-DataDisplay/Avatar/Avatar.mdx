# Avatar

Avatars can be used to represent people or objects. It supports images, `Icon`s, or letters.

## Examples

##### Basic

Three sizes and two shapes are available.

<CustomCanvas content={Examples.Basic} source={Sources.Basic} id="avatar-basic" />

##### Type

Image, Icon, letter and text are supported.

<CustomCanvas content={Examples.Type} source={Sources.Type} id="avatar-type" />

##### Initials

When property `label` is set, the first letter of the first word and the first letter of the second word are used. The default number of letters used is 2 but it can be changed using the `initialsMaxChars` property.

<CustomCanvas content={Examples.Initials} source={Sources.Initials} id="avatar-initials" />

##### Color

Letter and text can have colors and background colors.
When using custom color value, text color is automatically selected to allow optimal contrast level.
If the given color is invalid, the default color will be applied.

<CustomCanvas content={Examples.Color} source={Sources.Color} id="avatar-color" />

##### Avatar.Group

Avatar group display.

<CustomCanvas content={Examples.Group} source={Sources.Group} id="avatar-group" />

##### With Badge

Usually used for reminders and notifications.

<CustomCanvas content={Examples.Badge} source={Sources.Badge} id="avatar-badge" />

##### Responsive Size

Avatar size can be automatically adjusted based on the screen size.

<CustomCanvas content={Examples.Responsive} source={Sources.Responsive} id="avatar-responsive" />
