# Avatar

Avatars can be used to represent people or objects. It supports images, `Icon`s, or letters.

## Usage

Obviously (ðŸ¤­) when you need to display an avatar.

##### How to import
```jsx
import { KitAvatar } from "@kit/DataDisplay";

<KitAvatar.Group>
    <KitAvatar icon={<UserOutlined />} />
</KitAvatar.Group>
```
