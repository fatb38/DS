# Card

A component to display product informations

## Usage

The Card is used to display a product identity Card - some basic informations - such as a preview, label colors and so on.
It's also provides ability to select and / or edit product data

##### How to import
```jsx
import { KitCard, KitImage } from "@kit/DataDisplay/";

<KitCard picture={<KitImage src="images/tondeuse.png" />} />
```
