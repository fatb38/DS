# ItemCard

Component to display information with image and actions.

## Examples

### Basic

This is a basic ItemCard with a title. ItemCard has two display options: `card` and `list`.

<CustomCanvas content={Examples.Basic} source={Sources.Basic} id="item-card-basic" />

### With image

ItemCard can have an image. Use `imageSrc` property to set image source and `imageAlt` to set image alt text.

<CustomCanvas content={Examples.Image} source={Sources.Image} id="item-card-image" />

### With icon

ItemCard can display an Icon as an image. Use `icon` property to set icon. If `Ã¬mageSrc` is set, it will be displayed instead of the icon.

### Draggable

ItemCard can be dragged and dropped inside a list. To mark an **ItemCard** as draggable, add the `draggableHandler` prop.

Note that an icon appears and cursor changes, but you need to implement your own drag and drop solution in your application.
