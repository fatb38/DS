# ItemList

Display a row of a list.

## Usage

Use the **ItemList** to display a repeated piece of information from an unordered list.

## Examples

### Basic

A basic **ItemList** should contain a [`<KitIdCard />`](/docs/design-system-datadisplay-idcard--docs). You should provide `idCardProps`.

<CustomCanvas content={Examples.Basic} source={Sources.Basic} id="itemlist-basic" />

### Selectable

Each **ItemList** could be selectable if props `onSelect` is set.
When `onClick` props is also set, the select behavior is applied only on the checkbox.

> `onSelect` props will tell you if the item is selected or not.

<CustomCanvas content={Examples.Selectable} source={Sources.Selectable} id="itemlist-selectable" />

### Content

**ItemList** support extra content. The [`<KitIdCard />`](/docs/design-system-datadisplay-idcard--docs) will grow and the `content` props will fit to the content.

<CustomCanvas content={Examples.Content} source={Sources.Content} id="itemlist-content" />

### IdCardSubstitute

If needed, you can provide a custom card component via the `idCardSubstitute` property to replace the default `IdCard`.

<CustomCanvas content={Examples.IdCardSubstitute} source={Sources.IdCardSubstitute} id="itemlist-idcard-substitute" />

### Actions

**ItemList** can have many behaviors.

-   You can provide a method via props `onClick` that will be called when user click on it.
-   You can provide a list of actions buttons (`actions` props) that will be stacked on the right of the component.

<CustomCanvas content={Examples.Actions} source={Sources.Actions} id="itemlist-actions" />

### Draggable

**ItemList** can be dragged and dropped inside the list. To mark an **ItemList** as draggable, add the `draggable` props.

Note that an icon appears and cursor changes, but you need to implement your own drag and drop solution in your application.

<CustomCanvas content={Examples.Draggable} source={Sources.Draggable} id="itemlist-draggable" />

### All

You can combine all props together.

<CustomCanvas content={Examples.All} source={Sources.All} id="itemlist-all" />

### Disabled

You can disable `ItemList` to prevent all actions by setting the `disabled`.\
⚠️ ItemList doesn't apply `disabled` on the `content` and the `draggableHandler` props, you have to manage it by yourself.

<CustomCanvas content={Examples.Disabled} source={Sources.Disabled} id="itemlist-disabled" />
