# ItemList

Simple rectangular container with text, icons, and images and components such as Checkbox and Rafter, among others.

## Usage

Use the ItemList to display a repeated piece of information in, for instance, a List component, to allow users to take possible actions.

## Examples

A basic ItemList can contain a `title` and/or a `description`.
