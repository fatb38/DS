# Tag

Tag for categorizing or markup.

## Usage

-   It can be used to tag by dimension or property.

-   When categorizing.

## Examples

##### Basic

Usage of basic Tag, and it could be closable by set closable property. Closable Tag supports onClose events.

<CustomCanvas content={Examples.Basic} source={Sources.Basic} id="tag-basic" />

##### IdCard

Tag is using [`<KitIdCard />`](/docs/design-system-datadisplay-idcard--docs) component to display its contents. You can set `idCardProps.avatarProps` and `idCardProps.description` to the Tag.

<CustomCanvas content={Examples.IdCard} source={Sources.IdCard} id="tag-id-card" />

##### Type

Tag has 5 different types: `primary`, `secondary`, `neutral`, `error` and `success`.

<CustomCanvas content={Examples.Type} source={Sources.Type} id="tag-type" />

##### Disabled

To mark a Tag as disabled, add the `disabled` property to the Tag.

<CustomCanvas content={Examples.Disabled} source={Sources.Disabled} id="tag-disabled" />

##### Clickable

Setting `onClick` makes Tag clickable.

<CustomCanvas content={Examples.Clickable} source={Sources.Clickable} id="tag-clickable" />

##### Tag Group

Tag Group exemple

### TagGroup

| Property          | Description                                                  | Type            | Default |
| ----------------- | ------------------------------------------------------------ | --------------- | ------- |
| className         | The additionnal css class                                    | string          | -       |
| style             | The additionnal style                                        | CSSProperties   | -       |
| tags              | List of `KitTag` to display                                  | IKitTagConfig[] | -       |
| othersTagType     | Type of others tag displayed when there is to much tags      | KitTagType      | -       |
| othersTagDisabled | Disabled the others tag displayed when there is to much tags | ReactNode       | -       |
