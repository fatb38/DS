# Alert

`Alert` component for feedback.

## Usage

- When you need to show alert messages to users.

- When you need a persistent static container which is closable by user actions.

##### How to import
```jsx
import { KitAlert } from "@kit/Feedback/";

<KitAlert type="success" message="Success Text"/>
```

## Examples

##### Basic

The simplest usage for short messages.

{
<Canvas withSource="none">
    <Fragment>
        <KitAlert type="success" message="Success Text"/>
    </Fragment>
</Canvas>
}

##### More types

There are 4 types of Alert: `success`, `info`, `warning`, `error`.

{
<Canvas withSource="none">
    <Fragment>
        <KitAlert type="success" message="Success Text"/>
        <br/>
        <KitAlert type="info" message="Info Text"/>
        <br/>
        <KitAlert type="warning" message="Warning Text"/>
        <br/>
        <KitAlert type="error" message="Error Text"/>
    </Fragment>
</Canvas>
}

##### Closable

To show close button.

{
<Canvas withSource="none">
    <Fragment>
        <KitAlert type="warning" message="Warning Text" closable/>
        <br/>
        <KitAlert type="error" message="Error Text" closable/>
    </Fragment>

}
