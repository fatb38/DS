# How to use

## Import and usage

```tsx
import {useKitTheme} from 'aristid-ds';

const {theme} = useKitTheme();

const primaryColor = theme.colors.primary.blue100;
const backgroundColor = theme.colors.neutral.background;
const secondaryRedColor = theme.colors.secondary.error.red100;
```
