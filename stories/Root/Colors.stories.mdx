# How to use

## Import

```tsx
import theme from "/libs/components/theme";
```

## Usage

```tsx
const primaryColor = theme.color.primary.blue100;
const backgroundColor = theme.color.neutral.background;
const secondaryRedColor = theme.color.secondary.red.red100;
```
