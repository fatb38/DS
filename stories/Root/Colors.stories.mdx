# How to use    

## Import and usage

```tsx
import {useKitTheme} from 'aristid-ds';

const {theme} = useKitTheme();

const primaryColor = theme.general.colors.primary.primary100;
const backgroundColor = theme.general.colors.neutral.background;
const secondaryRedColor = theme.general.colors.secondary.red.red100;
```
